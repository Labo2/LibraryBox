Inspired by the PirateBox movement, in the spirit of pirate radio and free culture, LibraryBox is a self-contained digital resource sharing device. LibraryBox utilizes Free, Libre and Open Source software (FLOSS) to create mobile wireless file access networks where users can download digital content.

PirateBox, and thus LibraryBox, is designed to be safe and secure. No logins are required and no user data is logged. The system is purposely not connected to the Internet in order to prevent tracking and preserve user privacy.

PirateBox was created by David Darts and is registered under a Free Art License (2011). To learn more about the project or to find out how to build your own PirateBox system, please visit http://wiki.daviddarts.com/PirateBox

LibraryBox is a fork of PirateBox for the TP-Link MR 3020, customized to be friendly to Library needs. LibraryBox was conceived and created by Jason Griffey. More information can be found at the LibraryBox homepage
